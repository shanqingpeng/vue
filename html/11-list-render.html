1、使用 v-for 遍历数组 (使用 in) (常用)

  • ID ------- 姓名 ------- 年龄
  • {{person.id}} ------- {{person.name}} ------- {{person.age}}

2、使用 v-for 遍历数组 (使用 of)

  • ID ------- 姓名 ------- 年龄
  • {{person.id}} ------- {{person.name}} ------- {{person.age}}

3、使用 v-for 遍历对象 (可以使用 in 或 of) (不常用)

  • {{key}} ------- {{value}}

4、使用 v-for 遍历字符串 (可以使用 in 或 of) (不常用)

  • {{index}} ------- {{value}}

5、使用 v-for 指定遍历次数 (可以使用 in 或 of) (不常用)

  • {{index}} ------- {{value}}
